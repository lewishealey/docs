## Placement
- Pills appear in lists or rows should have an 8px where adjacent.
- **DON'T** mix the size of pills in a list or row.
- **DO** order pills by solids first, then strokes.

## Interaction

Static pills do not have interactions.

## Animation

Static pills do not have animations.
